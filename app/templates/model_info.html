Technical Details of the Skin Cancer Classifier

Dataset, training setup, model architecture and performance metrics

← Back to main page


Dataset

This model was trained using the public dataset HAM10000, which contains more than 10,000 dermatoscopic images of skin lesions categorized into seven classes:

  • AKIEC: Actinic keratoses
  • BCC: Basal cell carcinoma
  • BKL: Benign keratosis-like lesions
  • DF: Dermatofibroma
  • MEL: Melanoma
  • NV: Melanocytic nevi
  • VASC: Vascular lesions

Training Environment

The model was trained in Python using TensorFlow 2.10.0 with GPU acceleration. It was deployed inside a Docker container based on the official TensorFlow GPU image. The environment was set up to install dependencies from a requirements.txt file and serve the application via FastAPI and Uvicorn.


Model Architecture

The base model is EfficientNetV2S, selected after experimentation with different architectures available in the project Skin Cancer XAI - Universidad .

EfficientNetV2S is a convolutional neural network optimized for both speed and accuracy, with a total of 20.5 million parameters.

Calibration

To improve reliability in medical contexts, the model was calibrated using Temperature Scaling (T-scaling). The optimal temperature parameter was determined by minimizing the Expected Calibration Error (ECE), ensuring that the predicted probabilities better match the actual likelihood of each class.
